## 📌 Project 2: Stock Portfolio Tracker

### 🧠 Description
A Python-based application that allows users to calculate their total investment in stocks by entering stock names and quantities.

---

### ⚙️ Features
- Supports multiple stock entries
- Uses a dictionary to store stock prices
- Calculates total investment dynamically
- Handles invalid stock inputs
- Saves the result to a file (`portfolio.txt`)
- User-friendly exit system

---

### 🛠️ Technologies Used
- Python
- VS Code
